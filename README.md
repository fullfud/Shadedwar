# Fullfud Shahed Mod

Remote-controlled Shahed drone gameplay module for Forge 1.20.1. The project includes the entity, client monitor UI, custom audio, GeckoLib model, and networking stack needed to link, fly, and detonate the drone.

## Building

```bash
.\gradlew.bat build
```

The resulting distributable jar is produced under `build/libs/`.
